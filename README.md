Website portfolio profesional untuk **Putri Sari Azhari**, seorang Mining Control & Monitoring Engineer dengan 2 tahun pengalaman di industri pertambangan batubara.

## 🎨 Fitur Website

✨ **Design Modern & Professional**
- Dark theme dengan aksen biru cyan gradient
- Responsive design (mobile, tablet, desktop)
- Smooth animations dan transisi
- Performa optimal

📑 **Sections Lengkap**
- **Hero** - Intro dengan call-to-action buttons

## 🌐 Deploy ke GitHub Pages

1. Pergi ke repository settings
2. Scroll ke section **Pages**
3. Pilih **Deploy from branch**
4. Pilih **Branch: main** dan **Folder: / (root)**
5. Klik **Save**
6. Website akan live di: `https://bulcek.github.io/putri-portfolio`

## ✏️ Cara Customize

### Edit Informasi Personal
Buka `index.html` dan ubah:
- Nama: `<h1 class="hero-title">Putri Sari Azhari</h1>`
- Email: `putri.sari.azhari@email.com`
- Telepon: `+62 812-3456-7890`
- LinkedIn: `https://linkedin.com/in/putrisariazhari`

### Edit Warna Theme
Buka `styles.css` dan ubah CSS variables:
```css
:root {
    --primary-color: #0a1628;      /* Warna background utama */
    --accent-color: #00d4ff;       /* Warna accent (biru cyan) */
    --success-color: #00ff88;      /* Warna success (hijau) */
}
```

### Edit Konten Sections
- **About**: Ubah text di section `#about`
- **Skills**: Tambah/hapus skill cards
- **Experience**: Update timeline pengalaman
- **Contact**: Ubah info kontak
